Домашнее задание #6
Формы. Базы данных.
1. Используя форму, отправляем post запрос с данными: имя пользователя, фамилия, почта, номер телефона, на тот же адрес по которому сервер возвращает форму. Данные из формы валидируем  на сервере. Все поля обязательные. Сервер возвращает сообщение в виде html блока под формой, в котором зеленым цветом сообщение “Success” если форма валидна, красным “field_name required” если не валидна.
2. Форму из первого задания заполнить данными пользователя при ответе сервера с ошибкой. Чтоб пользователь заново не заполнял каждое поле, а только то, которое не валидно.
Для этого посмотреть документацию html form и изучить существующие атрибуты.
