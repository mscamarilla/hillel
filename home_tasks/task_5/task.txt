Домашнее задание #5
Включение файлов. Базовые теги HTML
1. Почитать больше о html, например здесь -> http://htmlbook.ru/samhtml
2. В индексе вывести ссылку (Show users) на другую страницу.
3. На странице по ссылке show users подключить файл в котором функция с одним аргументом, многомерным массивом. Каждый подмассив содержит информацию о пользователе. Функция возвращает html-шаблон таблицы с данными из входящего массива. Из индекса функцию вызвать с передачей массива из 10 пользователей и вернуть результат в браузер. Так же вместе с таблицей вернуть заголовок страницы.
4. Добавить в подключенный файл еще одну функцию. Она принимает тот же массив, но возвращает отформатированный список пользователей ( Данные пользователя конкатенировать). Смотри тег <ul><li>.
5. Положить в директорию images/, на сервере, файл-изображение. Добавить функцию  в подключенный файл, которая принимает тот же массив, но возвращает html блок с шапкой содержащей данные пользователя, а под ними изображение, которое мы ранее загрузили на сервер.
